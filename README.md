# Customer Behavior Analysis

## Project Goal
Analyze customer transactions and behavior to identify:
- Customer segments
- Purchase patterns
- Churn risks

## Technologies Used
- Python
- Pandas
- NumPy
- Seaborn
- Matplotlib
- Scikit-learn
- Google Colab

## Techniques Used
- Data Cleaning
- Feature Engineering
- RFM Analysis
- K-Means Clustering
- Data Visualization

## Key Insights
- Identified high-value customers
- Detected churn-risk customers
- Analyzed purchase trends and retention

## Recommendations
- Personalized marketing
- Loyalty programs
- Churn prevention campaigns
- Seasonal offers
- Product bundling
